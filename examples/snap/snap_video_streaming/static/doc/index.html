This snap manages the byte-range streaming protocol to stream videos to any web browser.
To stream a video, upload it in your library, then you can access it at the url: /video/your_video.mp4
an Example:
<video controls>
 <source src="/video/my_video.mp4" type="video/mp4">
 <source src="/video/my_video.webm" type="video/webm">
 <p>You web browser does not support html5 video</p>
</video>
